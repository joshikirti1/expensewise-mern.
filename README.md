💰 ExpenseWise – MERN Stack Expense Tracker

ExpenseWise is a full-stack expense tracking web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). It helps users manage and monitor their daily expenses efficiently through a simple and interactive interface.

The application allows users to add, view, and manage expenses, making it easier to track spending habits and maintain financial discipline.

🚀 Tech Stack

Frontend: React.js

Backend: Node.js, Express.js

Database: MongoDB (MongoDB Atlas)

Version Control: Git & GitHub

✨ Features

➕ Add new expenses

📋 View all expenses in a structured list

💾 Store expense data in MongoDB Atlas

⚡ Fast and responsive React UI

🔗 REST API integration between frontend and backend

🌐 Full-stack MERN architecture

📂 Project Structure
ExpenseWise
│
├── client (React Frontend)
├── server (Node.js + Express Backend)
└── database (MongoDB Atlas)
⚙️ How to Run the Project

Clone the repository

Install dependencies

Start backend server

Start React frontend

npm install
node server.js
npm start
🎯 Purpose

This project was built to practice full-stack development using the MERN stack, including frontend UI development, backend API creation, and cloud database integration.
